# STORY-260309-kmus9f: capability-presets

## Description
Provide high-level capability presets that bundle related entitlements + Tuist capabilities + config changes. Examples: push-notifications (aps-environment + .pushNotifications capability + background-modes remote-notification), app-groups (com.apple.security.application-groups + .appGroups capability), keychain-sharing (keychain-access-groups + .keychainSharing). Single command like: ios-app-manager capability add push-notifications --env development.
