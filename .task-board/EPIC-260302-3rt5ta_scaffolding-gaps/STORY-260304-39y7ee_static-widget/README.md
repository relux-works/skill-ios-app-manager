# STORY-260304-39y7ee: static-widget

## Description
Scaffold a static widget plugin that registers into widget-base WidgetBundle.

What it creates:
1. StaticConfiguration widget struct
2. TimelineProvider (placeholder/snapshot/timeline)
3. TimelineEntry model
4. Widget View stub (SwiftUI)
5. Registration into WidgetBundle from widget-base

This is the simplest widget type — no intents, no interactivity. Pure timeline-driven snapshots.

Depends on: widget-base
