# STORY-260714-1mpsae: release-tagging-and-changelog

## Description
Cut the next skill-ios-app-manager release from `main`, document the delta since `v0.10.0`, and publish matching release metadata.

## Scope
Review commits since `v0.10.0`, write a repository changelog entry, verify the release candidate, and push the release tag plus GitHub release notes.

## Acceptance Criteria
- Changelog entry accurately summarizes the delta since `v0.10.0`.
- Release tag is created from the verified `main` commit.
- GitHub release notes reflect the changelog entry.
