# EPIC-260225-2w1bvf: ioc-relux-integration

## Description
Add SwiftIoC and swift-relux setup commands to ios-app-manager CLI. Two separate features: (1) ioc setup — adds SwiftIoC dependency, creates IoC registry in the app target, wires module registration; (2) relux setup — adds swift-relux + swiftui-relux dependencies, creates Relux bootstrap, connects modules to store. Each is a separate CLI command with scaffold templates.
