# STORY-260224-1lqyng: entitlements-management

## Description
CRUD operations for iOS entitlements via CLI. Add/remove/list/update entitlements (push notifications, app groups, keychain sharing, iCloud, HealthKit, etc). Modifies the entitlements plist and keeps Tuist project config in sync. Commands: tuist-starter entitlements add, tuist-starter entitlements remove, tuist-starter entitlements list.
