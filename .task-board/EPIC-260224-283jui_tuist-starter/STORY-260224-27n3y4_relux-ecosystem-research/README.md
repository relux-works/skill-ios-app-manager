# STORY-260224-27n3y4: relux-ecosystem-research

## Description
Research relux ecosystem packages to understand architecture, patterns, and conventions for relux-manager component. Relux = Redux/Flux-inspired async-first Swift framework. All feature modules inside Tuist project are built on Relux (except utility modules). Packages: swift-relux (core), swiftui-relux (SwiftUI bindings), swift-ioc (DI container), swiftui-reluxrouter (navigation), relux-sample (reference app), swift-httpclient (HTTP client).
