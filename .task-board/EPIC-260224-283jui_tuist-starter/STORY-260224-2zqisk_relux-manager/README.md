# STORY-260224-2zqisk: relux-manager

## Description
Third CLI component: relux-manager. Scaffolds and manages Relux module internals. When a new feature module is created via tuist-proj-manager, relux-manager generates the Relux boilerplate inside: Store, Reducer, Actions, State, Middleware, View bindings. Templates based on patterns discovered in relux ecosystem forensics. Commands: relux-manager init (bootstrap Relux structure inside a module), relux-manager add-action, add-reducer, add-middleware, etc. Utility modules skip Relux scaffolding.
